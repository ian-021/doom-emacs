# Doom Emacs Configuration

Personal Doom Emacs configuration files.

## Files

- `init.el` - Doom modules configuration
- `config.el` - Personal configuration and customizations
- `packages.el` - Additional packages

## Features

- Gruvbox theme
- Relative line numbers
- Custom terminal toggle with `C-c t`
- vterm terminal emulator (30% window height at bottom)
- Evil mode disabled in terminal for normal terminal functionality

## Installation

1. Install [Doom Emacs](https://github.com/doomemacs/doomemacs)
2. Clone this repository to `~/.config/doom/`
3. Run `doom sync`
4. Restart Emacs

## Key Bindings

- `C-c t` - Toggle terminal at bottom
